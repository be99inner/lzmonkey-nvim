# My Personal Configuration for Neovim

Requires

- Neovim version more than 0.8

## Features

- UI configuration
- Motivation Improvements
- Tools Integration
- LSP configuration

## Installation

```bash
mv ~/.config/nvim ~/.config/nvim_bak
git clone git@github.com:be99inner/lzmonkey-nvim.git ~/.config/nvim
```
